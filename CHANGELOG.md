# Changelog

Все значимые изменения в проекте LootBay будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Система ролей пользователей (OWNER, ADMIN, MODERATOR, SUPPORT)
- Ролевые кнопки в заголовке сайта
- API endpoints для управления ролями
- Автоматическое обновление JWT токенов при отсутствии ролей
- Документация по системе ролей (ROLES_SYSTEM.md)

### Changed
- Обновлена структура JWT токенов для включения ролей
- Улучшен UI заголовка с ролевыми элементами
- Оптимизирована работа с токенами на фронтенде

### Fixed
- Исправлена маршрутизация API Gateway для ролевых endpoints
- Устранены проблемы с GORM automigration
- Исправлена обработка ролей в middleware

## [1.0.0] - 2024-01-XX

### Added
- Микросервисная архитектура на Go
- Next.js фронтенд с TypeScript
- Система аутентификации через JWT
- API Gateway с проксированием запросов
- Сервисы: User, Product, Order, Payment, Chat, Notification
- Базы данных: PostgreSQL, Redis, MongoDB
- Docker контейнеризация
- Мониторинг через Prometheus и Grafana
- Система миграций базы данных
- Загрузка аватаров пользователей
- Система отзывов и рейтингов
- Чат между пользователями
- Система уведомлений
- Кошелек и транзакции
- Управление товарами и заказами
- Категории и игры
- Постоянные аккаунты пользователей

### Security
- Хеширование паролей с bcrypt
- Rate limiting
- CORS настройки
- Валидация входных данных
- Защита от SQL injection

---

## Типы изменений

- **Added** - новая функциональность
- **Changed** - изменения в существующей функциональности
- **Deprecated** - функциональность, которая будет удалена
- **Removed** - удаленная функциональность
- **Fixed** - исправления багов
- **Security** - исправления уязвимостей
